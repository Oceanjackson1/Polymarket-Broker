# 分析 API

基础路径：`/api/v1/analysis`

所需权限作用域：`analysis:read`

分析 API 提供基于 DeepSeek 的 AI 分析，覆盖单个市场、NBA 比赛及全市场扫描。所有用户（包括自有应用和第三方开发者）同步接收信号，无内部优先队列。

**免费套餐配额：** 所有 `/analysis/` 端点共享每日 10 次调用上限。

## 端点列表

| 方法 | 路径 | 描述 |
|---|---|---|
| `GET` | `/analysis/market/\{market_id\}` | AI 概率估算 vs 市场价格 |
| `POST` | `/analysis/scan` | 全市场定价偏差扫描 |
| `GET` | `/analysis/nba/\{game_id\}` | NBA 比赛 AI 方向性建议 |
| `POST` | `/analysis/ask` | 自然语言查询 |

---

## GET /analysis/market/\{market_id\}

获取某市场结果的 AI 概率估算，并与当前市场价格进行对比。

**查询参数：** `token_id`（可选，默认为第一个结果）

**响应 `200`：**

```json
{
  "market_id": "0xabc123...",
  "market_title": "Will the LA Lakers win the 2026 NBA Championship?",
  "token_id": "217426331",
  "outcome_label": "Yes",
  "market_price": 0.12,
  "ai_estimate": 0.09,
  "bias_direction": "MARKET_OVERPRICED",
  "bias_bps": 300,
  "confidence": "medium",
  "reasoning": "Lakers are currently 10th in the Western Conference standings with a -3.2 net rating. Historical base rate for a sub-.500 team winning the championship is approximately 8-10%. Current market price of 12% appears slightly elevated.",
  "model": "deepseek-r1",
  "generated_at": "2026-03-19T14:22:05Z"
}
```

| 字段 | 描述 |
|---|---|
| `bias_direction` | `MARKET_OVERPRICED` — AI 给出的概率低于市场。`MARKET_UNDERPRICED` — AI 给出的概率高于市场。`NEUTRAL` — 差距在 300 个基点以内。 |
| `bias_bps` | 绝对差值（单位：基点） |
| `confidence` | `low`（低）/ `medium`（中）/ `high`（高） |
| `reasoning` | AI 估算的自然语言解释 |

## POST /analysis/scan

全市场扫描，返回定价偏差最大的机会。可用于发现所有类别中定价最不合理的市场。

**请求体：**

```json
{
  "categories": ["sports", "crypto"],
  "min_volume_24h": 10000,
  "min_bias_bps": 500,
  "limit": 20
}
```

所有字段均为可选。默认扫描所有活跃市场，`min_bias_bps` 默认为 300。

## POST /analysis/ask

提交关于任意市场或主题的自然语言查询。

**请求体：**

```json
{
  "query": "What's the best opportunity in NBA markets right now?",
  "context": {
    "include_live_games": true,
    "include_market_data": true
  }
}
```

**响应 `200`：**

```json
{
  "query": "What's the best opportunity in NBA markets right now?",
  "answer": "Based on current game states and Polymarket odds, the most attractive opportunity is Golden State Warriors in the GSW vs LAL game currently in the 3rd quarter. The model estimates GSW at 35% win probability while the market prices them at 31%, representing a 400bps gap. Volume is strong at $87k today, providing reasonable liquidity for a position up to $5,000. The suggested entry: BUY Golden State Warriors outcome at 0.31 with a target of 0.35 (if the market corrects) or 1.0 (on resolution).",
  "referenced_markets": [
    {
      "market_id": "0xabc123...",
      "title": "GSW vs LAL - March 19",
      "current_price": 0.31
    }
  ],
  "model": "deepseek-r1",
  "generated_at": "2026-03-19T14:22:05Z"
}
```
