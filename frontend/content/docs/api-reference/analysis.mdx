# Analysis API

Base path: `/api/v1/analysis`

Scope required: `analysis:read`

The Analysis API provides DeepSeek-powered AI analysis for individual markets, NBA games, and full-market scans. All consumers (own apps and third-party developers) receive signals simultaneously — no internal priority queue.

**Free tier quota:** 10 calls/day shared across all `/analysis/` endpoints.

## Endpoints

| Method | Path | Description |
|---|---|---|
| `GET` | `/analysis/market/\{market_id\}` | AI probability estimate vs market price |
| `POST` | `/analysis/scan` | Full-market scan for pricing bias |
| `GET` | `/analysis/nba/\{game_id\}` | AI directional suggestion for NBA game |
| `POST` | `/analysis/ask` | Natural language query |

---

## GET /analysis/market/\{market_id\}

Get an AI probability estimate for a market outcome and compare it to the current market price.

**Query parameters:** `token_id` (optional, defaults to first outcome)

**Response `200`:**

```json
{
  "market_id": "0xabc123...",
  "market_title": "Will the LA Lakers win the 2026 NBA Championship?",
  "token_id": "217426331",
  "outcome_label": "Yes",
  "market_price": 0.12,
  "ai_estimate": 0.09,
  "bias_direction": "MARKET_OVERPRICED",
  "bias_bps": 300,
  "confidence": "medium",
  "reasoning": "Lakers are currently 10th in the Western Conference standings with a -3.2 net rating. Historical base rate for a sub-.500 team winning the championship is approximately 8-10%. Current market price of 12% appears slightly elevated.",
  "model": "deepseek-r1",
  "generated_at": "2026-03-19T14:22:05Z"
}
```

| Field | Description |
|---|---|
| `bias_direction` | `MARKET_OVERPRICED` — AI gives lower probability than market. `MARKET_UNDERPRICED` — AI gives higher probability. `NEUTRAL` — within 300 bps. |
| `bias_bps` | Absolute difference in basis points |
| `confidence` | `low` / `medium` / `high` |
| `reasoning` | Natural language explanation of the AI estimate |

## POST /analysis/scan

Full-market scan returning the top pricing-bias opportunities. Useful for identifying the most mispriced markets across all categories.

**Request body:**

```json
{
  "categories": ["sports", "crypto"],
  "min_volume_24h": 10000,
  "min_bias_bps": 500,
  "limit": 20
}
```

## POST /analysis/ask

Submit a natural language query about any market or topic.

**Request body:**

```json
{
  "query": "What's the best opportunity in NBA markets right now?",
  "context": {
    "include_live_games": true,
    "include_market_data": true
  }
}
```

**Response `200`:**

```json
{
  "query": "What's the best opportunity in NBA markets right now?",

The model estimates GSW at 35% win probability while the market prices them at 31%, representing a 400bps gap. Volume is strong at $87k today, providing reasonable liquidity for a position up to $5,000. The suggested entry: BUY Golden State Warriors outcome at 0.31 with a target of 0.35 (if the market corrects) or 1.0 (on resolution).",

    {
      "market_id": "0xabc123...",
      "title": "GSW vs LAL - March 19",
      "current_price": 0.31
    }
  ],
  "model": "deepseek-r1",
  "generated_at": "2026-03-19T14:22:05Z"
}
```
