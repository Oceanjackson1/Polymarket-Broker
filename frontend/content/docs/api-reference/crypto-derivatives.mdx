# Crypto Derivatives

Base path: `/api/v1/data/crypto`

Real-time and historical crypto derivatives data sourced from CoinGlass. Covers BTC, ETH, SOL across major exchanges including Binance, Bybit, OKX, and dYdX.

## Endpoints

| Method | Path | Description | Auth |
|---|---|---|---|
| `GET` | `/data/crypto/funding-rates` | Aggregated funding rates (avg/min/max + per-exchange) | API Key |
| `GET` | `/data/crypto/funding-rates/history` | Historical funding rates | API Key |
| `GET` | `/data/crypto/open-interest` | Open interest total + changes across timeframes | API Key |
| `GET` | `/data/crypto/open-interest/history` | Open interest history | API Key |
| `GET` | `/data/crypto/liquidations` | Liquidation data (long/short breakdown) | API Key |
| `GET` | `/data/crypto/taker-volume` | Buy/sell ratio and taker volumes | API Key |
| `GET` | `/data/crypto/sentiment` | Fear & Greed index | API Key |
| `GET` | `/data/crypto/overview` | All-in-one summary of all crypto metrics | API Key |

---

## GET /data/crypto/funding-rates

Returns current aggregated funding rates across exchanges for a given symbol.

**Query parameters:**

| Param | Type | Required | Description |
|---|---|---|---|
| `symbol` | string | Yes | Trading symbol: `BTC`, `ETH`, or `SOL` |

**Example request:**

```bash
curl "https://api.polydesk.io/api/v1/data/crypto/funding-rates?symbol=BTC" \
  -H "X-API-Key: YOUR_KEY"
```

**Example response:**

```json
{
  "symbol": "BTC",
  "aggregated": {
    "avg_rate": 0.0103,
    "min_rate": 0.0051,
    "max_rate": 0.0200,
    "avg_annual_pct": 10.71
  },
  "exchanges": [
    { "exchange": "Binance", "rate": 0.0100, "annual_pct": 10.40 },
    { "exchange": "Bybit", "rate": 0.0120, "annual_pct": 12.48 },
    { "exchange": "OKX", "rate": 0.0089, "annual_pct": 9.26 }
  ],
  "timestamp": "2026-03-31T12:00:00Z"
}
```

## GET /data/crypto/funding-rates/history

Returns historical funding rate snapshots.

**Query parameters:**

| Param | Type | Required | Description |
|---|---|---|---|
| `symbol` | string | Yes | Trading symbol: `BTC`, `ETH`, or `SOL` |
| `period` | string | No | Lookback period: `1d`, `7d`, `30d` (default `7d`) |

**Example request:**

```bash
curl "https://api.polydesk.io/api/v1/data/crypto/funding-rates/history?symbol=ETH&period=7d" \
  -H "X-API-Key: YOUR_KEY"
```

**Example response:**

```json
{
  "symbol": "ETH",
  "period": "7d",
  "data": [
    { "timestamp": "2026-03-24T08:00:00Z", "avg_rate": 0.0085 },
    { "timestamp": "2026-03-24T16:00:00Z", "avg_rate": 0.0091 },
    { "timestamp": "2026-03-25T00:00:00Z", "avg_rate": 0.0078 }
  ]
}
```
