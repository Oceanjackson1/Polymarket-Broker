# BTC 数据 API

基础路径：`/api/v1/data/btc`

所需权限作用域：`data:read`

BTC 数据 API 提供多时间框架 BTC 预测市场数据、毫秒精度的 Polygon 链上交易数据以及历史回填数据。数据来源包括 BTC UpDown MultiTimeframe Trade Scraper、Dome API 及直接 Polygon 链索引。

## 端点列表

| 方法 | 路径 | 描述 |
|---|---|---|
| `GET` | `/data/btc/predictions` | 列出所有 BTC 预测市场 |
| `GET` | `/data/btc/predictions/\{timeframe\}` | 特定时间框架的预测数据 |
| `GET` | `/data/btc/onchain` | Polygon 链上交易数据 |
| `GET` | `/data/btc/history` | 通过 Gamma API 获取的历史回填数据 |

<Callout type="info">
  `/data/btc/onchain` 是顶级路由（不嵌套在 `/predictions/\{timeframe\}` 下），以避免与 `\{timeframe\}` 路径参数产生路由冲突。
</Callout>

## GET /data/btc/onchain

BTC 预测市场的 Polygon 链上交易数据，精度达毫秒级，直接来源于 Polygon 链索引。

**查询参数：**

| 参数 | 类型 | 默认值 | 描述 |
|---|---|---|---|
| `timeframe` | string | `all` | 按市场时间框架筛选：`5m`、`15m`、`1h`、`4h`、`all` |
| `from` | ISO 8601 | 1 小时前 | 开始时间 |
| `to` | ISO 8601 | 当前时间 | 结束时间 |
| `limit` | integer | 100 | 每页条数（最大 500） |
| `cursor` | string | — | 分页游标 |
| `min_size` | float | — | 最小交易金额（USDC） |

**响应 `200`：**

```json
{
  "data": [
    {
      "tx_hash": "0xtxhash...",
      "block_number": 67234891,
      "timestamp_ms": 1742394725432,
      "timestamp": "2026-03-19T14:22:05.432Z",
      "market_id": "0xbtc5m...",
      "timeframe": "5m",
      "token_id": "btc5m_yes",
      "outcome": "YES",
      "side": "BUY",
      "price": 0.52,
      "size": 500.0,
      "maker": "0xMakerAddress...",
      "taker": "0xTakerAddress...",
      "fee_amount": 0.26
    },
    {
      "tx_hash": "0xtxhash2...",
      "block_number": 67234890,
      "timestamp_ms": 1742394724891,
      "timestamp": "2026-03-19T14:22:04.891Z",
      "market_id": "0xbtc15m...",
      "timeframe": "15m",
      "token_id": "btc15m_yes",
      "outcome": "YES",
      "side": "BUY",
      "price": 0.55,
      "size": 1200.0,
      "maker": "0xMakerAddress...",
      "taker": "0xTakerAddress...",
      "fee_amount": 0.33
    }
  ],
  "pagination": {
    "cursor": "...",
    "has_more": true,
    "limit": 100
  },
  "data_updated_at": "2026-03-19T14:22:05Z"
}
```

毫秒精度的 `timestamp_ms` 字段可用于检测抢跑模式和高频订单流分析。

## GET /data/btc/history

通过 Polymarket Gamma API 获取的历史回填数据，返回已结算的 BTC 预测市场历史记录。

**查询参数：**

| 参数 | 类型 | 默认值 | 描述 |
|---|---|---|---|
| `timeframe` | string | `all` | `5m`、`15m`、`1h`、`4h`、`all` |
| `from` | ISO 8601 | 30 天前 | 开始日期 |
| `to` | ISO 8601 | 当前时间 | 结束日期 |
| `limit` | integer | 100 | 每页条数（最大 500） |
| `cursor` | string | — | 分页游标 |

**响应 `200`：**

```json
{
  "data": [
    {
      "market_id": "0xbtc5m_historical...",
      "timeframe": "5m",
      "opened_at": "2026-03-18T10:00:00Z",
      "resolved_at": "2026-03-18T10:05:00Z",
      "outcome": "YES",
      "entry_btc_price": 67800.00,
      "exit_btc_price": 67950.00,
      "change_pct": 0.22,
      "yes_resolution_price": 1.0,
      "no_resolution_price": 0.0,
      "volume": 34567.89
    }
  ],
  "pagination": { "cursor": "...", "has_more": true, "limit": 100 },
  "summary": {
    "total_resolved": 8640,
    "yes_count": 4385,
    "no_count": 4255,
```
