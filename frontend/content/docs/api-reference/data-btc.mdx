# BTC Data API

Base path: `/api/v1/data/btc`

Scope required: `data:read`

The BTC Data API provides access to multi-timeframe BTC prediction markets, on-chain Polygon trade data at millisecond precision, and historical backfill data. Sources include the BTC UpDown MultiTimeframe Trade Scraper, Dome API, and direct Polygon chain indexing.

## Endpoints

| Method | Path | Description |
|---|---|---|
| `GET` | `/data/btc/predictions` | List all BTC prediction markets |
| `GET` | `/data/btc/predictions/\{timeframe\}` | Prediction data for a specific timeframe |
| `GET` | `/data/btc/onchain` | On-chain Polygon trade data |
| `GET` | `/data/btc/history` | Historical backfill via Gamma API |

<Callout type="info">
  `/data/btc/onchain` is a top-level route (not nested under `/predictions/\{timeframe\}`) to avoid router collision with the `\{timeframe\}` path parameter.
</Callout>

## GET /data/btc/onchain

On-chain Polygon trade data for BTC prediction markets at millisecond precision. Sourced directly from Polygon chain indexing.

**Query parameters:**

| Parameter | Type | Default | Description |
|---|---|---|---|
| `timeframe` | string | `all` | Filter by market timeframe: `5m`, `15m`, `1h`, `4h`, `all` |
| `from` | ISO 8601 | 1 hour ago | Start time |
| `to` | ISO 8601 | Now | End time |
| `limit` | integer | 100 | Items per page (max 500) |
| `cursor` | string | — | Pagination cursor |
| `min_size` | float | — | Minimum trade size in USDC |

**Response `200`:**

## GET /data/btc/history

Historical backfill data via the Polymarket Gamma API. Returns resolved BTC prediction market history.

**Query parameters:**

| Parameter | Type | Default | Description |
|---|---|---|---|
| `timeframe` | string | `all` | `5m`, `15m`, `1h`, `4h`, `all` |
| `from` | ISO 8601 | 30 days ago | Start date |
| `to` | ISO 8601 | Now | End date |
| `limit` | integer | 100 | Items per page (max 500) |
| `cursor` | string | — | Pagination cursor |

**Response `200`:**

```json
{
  "data": [
    {
      "market_id": "0xbtc5m_historical...",
      "timeframe": "5m",
      "opened_at": "2026-03-18T10:00:00Z",
```
