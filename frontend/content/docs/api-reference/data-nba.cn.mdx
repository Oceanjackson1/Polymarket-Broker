# NBA 数据 API

基础路径：`/api/v1/data/nba`

所需权限作用域：`data:read`

NBA 数据 API 将实时 NBA 比赛数据（ESPN + NBA 官方数据源）与 Polymarket CLOB 实时赔率融合，生成偏差信号——识别 Polymarket 定价与比赛状态所隐含概率之间的偏差。

此数据无法通过 Polymarket 官方接口直接获取。NBA 数据采集器在比赛进行期间每 30 秒更新一次。

## 端点列表

| 方法 | 路径 | 描述 |
|---|---|---|
| `GET` | `/data/nba/games` | 今日及即将进行的 NBA 比赛 |
| `GET` | `/data/nba/games/\{game_id\}` | 含实时比分的比赛详情 |
| `GET` | `/data/nba/games/\{game_id\}/fusion` | 实时比分 × Polymarket 赔率 + 偏差信号 |
| `GET` | `/data/nba/games/\{game_id\}/orderbook` | 该比赛的 Polymarket 订单簿 |
| `WS` | `/ws/data/nba/\{game_id\}/live` | 实时比分 + 赔率同步 |

## GET /data/nba/games

列出今日及即将进行的 NBA 比赛及其关联的 Polymarket 市场。

**查询参数：**

| 参数 | 类型 | 默认值 | 描述 |
|---|---|---|---|
| `date` | `YYYY-MM-DD` | 今日（UTC） | 指定日期 |
| `status` | string | `all` | `scheduled`、`live`、`final`、`all` |

## GET /data/nba/games/\{game_id\}/fusion

核心旗舰端点。返回实时比赛状态与 Polymarket 当前隐含概率的交叉分析，以及计算出的偏差信号，用于指示定价偏离情况。

**响应 `200`：**

```json
{
  "game_id": "nba_2026_gsw_lal",
  "score": {
    "home": 87,
    "away": 94,
    "quarter": 3,
    "time_remaining": "4:22"
  },
  "polymarket": {
    "home_win_prob": 0.31,
    "away_win_prob": 0.69,
    "last_trade_price": 0.69,
    "spread": 0.02,
    "market_id": "0xabc123..."
  },
  "model": {
    "home_win_prob": 0.35,
    "away_win_prob": 0.65,
    "model_source": "in_game_win_probability"
  },
  "bias_signal": {
    "direction": "HOME_UNDERPRICED",
    "magnitude_bps": 420,
    "description": "Model gives GSW 35% chance; market prices them at 31%. 400bps gap."
  },
  "stale": false,
  "updated_at": "2026-03-17T22:14:05Z"
}
```

### 偏差信号

| 字段 | 描述 |
|---|---|
| `direction` | `HOME_UNDERPRICED` — 主队模型概率高于市场价格。`AWAY_UNDERPRICED` — 客队模型概率高于市场价格。`NEUTRAL` — 偏差低于 300 个基点阈值。 |
| `magnitude_bps` | 模型概率与市场概率之间的绝对差值（单位：基点） |

偏差信号阈值为 300 个基点，低于此阈值的信号将被归类为 `NEUTRAL`。

### 数据陈旧处理

若 NBA 数据采集器在 30 秒内未推送新数据，响应将包含 `"stale": true`。`updated_at` 字段始终显示最后一次刷新时间。实时交易信号的使用方应自行实现陈旧容忍机制。

## WS /ws/data/nba/\{game_id\}/live

实时 WebSocket 流，同步 NBA 实时比分更新与 Polymarket 赔率变化。

**连接地址：** `wss://api.broker.polymarket.com/ws/data/nba/\{game_id\}/live`

**比分更新消息：**

```json
{
  "type": "score_update",
  "game_id": "nba_2026_gsw_lal",
  "score": { "home": 89, "away": 94, "quarter": 3, "time_remaining": "3:45" },
  "timestamp": "2026-03-19T22:14:30Z"
}
```

**赔率更新消息：**

```json
{
  "type": "odds_update",
  "game_id": "nba_2026_gsw_lal",
  "home_win_prob": 0.32,
  "away_win_prob": 0.68,
  "bias_signal": {
    "direction": "HOME_UNDERPRICED",
    "magnitude_bps": 300
  },
  "timestamp": "2026-03-19T22:14:32Z"
}
```

WebSocket 连接不计入每日 API 调用配额。
