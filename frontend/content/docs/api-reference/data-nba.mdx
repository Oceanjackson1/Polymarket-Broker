# NBA Data API

Base path: `/api/v1/data/nba`

Scope required: `data:read`

The NBA Data API fuses live NBA game data (ESPN + NBA Official) with real-time Polymarket CLOB odds to produce bias signals — identifying when Polymarket pricing diverges from what game state implies.

This data is unavailable via Polymarket directly. The NBA collector updates every 30 seconds during live games.

## Endpoints

| Method | Path | Description |
|---|---|---|
| `GET` | `/data/nba/games` | Today's and upcoming NBA games |
| `GET` | `/data/nba/games/\{game_id\}` | Game detail with live score |
| `GET` | `/data/nba/games/\{game_id\}/fusion` | Live score × Polymarket odds + bias signal |
| `GET` | `/data/nba/games/\{game_id\}/orderbook` | Polymarket orderbook for this game |
| `WS` | `/ws/data/nba/\{game_id\}/live` | Real-time score + odds sync |

## GET /data/nba/games/\{game_id\}/fusion

The flagship endpoint. Returns live game state crossed with current Polymarket implied probabilities, plus a computed bias signal indicating pricing divergence.

**Response `200`:**

```json
{
  "game_id": "nba_2026_gsw_lal",
  "score": {
    "home": 87,
    "away": 94,
    "quarter": 3,
    "time_remaining": "4:22"
  },
  "polymarket": {
    "home_win_prob": 0.31,
    "away_win_prob": 0.69,
    "last_trade_price": 0.69,
    "spread": 0.02,
    "market_id": "0xabc123..."
  },
  "model": {
    "home_win_prob": 0.35,
    "away_win_prob": 0.65,
    "model_source": "in_game_win_probability"
  },
  "bias_signal": {
    "direction": "HOME_UNDERPRICED",
    "magnitude_bps": 420,
    "description": "Model gives GSW 35% chance; market prices them at 31%. 400bps gap."
  },
  "stale": false,
  "updated_at": "2026-03-17T22:14:05Z"
}
```

### Bias Signal

| Field | Description |
|---|---|
| `direction` | `HOME_UNDERPRICED` — model probability > market price for home team. `AWAY_UNDERPRICED` — model probability > market price for away team. `NEUTRAL` — divergence below 300 bps threshold. |
| `magnitude_bps` | Absolute difference in basis points between model and market probability |

The bias signal threshold is 300 bps. Signals below this are classified as `NEUTRAL`.

### Stale Data Handling

If the NBA collector has not pushed a fresh update within 30 seconds, the response includes `"stale": true`. The `updated_at` field always shows the last refresh time. Consumers should implement their own staleness tolerance for live trading signals.

## WS /ws/data/nba/\{game_id\}/live

Real-time WebSocket stream syncing live NBA score updates with Polymarket odds changes.

**Connection:** `wss://api.broker.polymarket.com/ws/data/nba/\{game_id\}/live`

**Score update message:**

```json
{
  "type": "score_update",
  "game_id": "nba_2026_gsw_lal",
  "score": { "home": 89, "away": 94, "quarter": 3, "time_remaining": "3:45" },
  "timestamp": "2026-03-19T22:14:30Z"
}
```

**Odds update message:**

```json
{
  "type": "odds_update",
  "game_id": "nba_2026_gsw_lal",
  "home_win_prob": 0.32,
  "away_win_prob": 0.68,
  "bias_signal": {
    "direction": "HOME_UNDERPRICED",
    "magnitude_bps": 300
  },
  "timestamp": "2026-03-19T22:14:32Z"
}
```

WebSocket connections do not count against the daily API call quota.
