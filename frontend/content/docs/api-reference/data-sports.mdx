# Sports Data API

Base path: `/api/v1/data/sports`

Scope required: `data:read`

The Sports Data API provides enhanced orderbook and event data for 145 sport and esport categories. This includes full historical order books per event â€” data that is unavailable via Polymarket directly.

## Endpoints

| Method | Path | Description |
|---|---|---|
| `GET` | `/data/sports/categories` | List all supported sport categories |
| `GET` | `/data/sports/\{sport\}/events` | Events for a specific sport |
| `GET` | `/data/sports/\{sport\}/events/\{event_id\}/orderbook` | Historical orderbook for an event |
| `GET` | `/data/sports/\{sport\}/events/\{event_id\}/realized` | Realized trade history |

## Sport Identifier Format

Sports are identified by URL-safe slug strings. The full list is returned by `/data/sports/categories`.

**Examples:** `nba`, `nfl`, `epl`, `ufc`, `cs2`, `lol`, `nhl`, `mlb`, `atp`, `wta`

## GET /data/sports/\{sport\}/events/\{event_id\}/orderbook

Get the full historical order book for a specific event. This endpoint is the key differentiator â€” Polymarket's own API only returns the current state, not the full time series.

**Query parameters:**

| Parameter | Type | Default | Description |
|---|---|---|---|
| `token_id` | string | â€” | Outcome token ID |
| `from` | ISO 8601 | Event start | Start of history window |
| `to` | ISO 8601 | Now | End of history window |
| `interval` | string | `1m` | Snapshot interval: `1s`, `1m`, `5m`, `1h` |

**Response `200`:**

```json
{
  "event_id": "sports_nba_gsw_lal_20260319",
  "token_id": "217426331",
  "interval": "1m",
  "stale": false,
  "data_updated_at": "2026-03-19T14:20:00Z",
  "snapshots": [
    {
      "timestamp": "2026-03-19T14:00:00Z",
      "bids": [
        { "price": 0.44, "size": 500.0 },
        { "price": 0.43, "size": 1200.0 }
      ],
      "asks": [
        { "price": 0.45, "size": 300.0 },
        { "price": 0.46, "size": 800.0 }
      ],
      "mid": 0.445
    },
    {
      "timestamp": "2026-03-19T14:01:00Z",
      "bids": [...],
      "asks": [...],
      "mid": 0.448
    }
  ]
}
```

<Callout type="info">
  If the data collector hasn't pushed a fresh update within 5 minutes, the response will include `"stale": true`. The `data_updated_at` field always shows the last refresh time.
</Callout>
