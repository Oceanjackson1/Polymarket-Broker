# 市场 API

基础路径：`/api/v1/markets`

所需权限作用域：`markets:read`

市场 API 提供对所有 Polymarket 事件的访问，涵盖政治、体育、金融、加密货币、科技等各类别。数据来源于 Polymarket Gamma API、CLOB API 以及实时抓取。

## 端点列表

| 方法 | 路径 | 描述 |
|---|---|---|
| `GET` | `/markets` | 列出所有市场（支持筛选） |
| `GET` | `/markets/\{market_id\}` | 市场详情 |
| `GET` | `/markets/\{market_id\}/orderbook` | 实时订单簿 |
| `GET` | `/markets/\{market_id\}/trades` | 历史成交记录（分页） |
| `GET` | `/markets/\{market_id\}/midpoint` | 中间价 |
| `GET` | `/markets/search` | 全文搜索 |
| `WS` | `/ws/markets/\{market_id\}` | 实时订单簿流 |

---

## GET /markets

列出所有市场，支持按类别、状态和标签筛选。

**查询参数：**

| 参数 | 类型 | 默认值 | 描述 |
|---|---|---|---|
| `category` | string | — | 按事件类别筛选（如 `sports`、`politics`、`crypto`） |
| `status` | string | `active` | 按状态筛选：`active`、`resolved`、`all` |
| `tag` | string | — | 按标签 slug 筛选 |
| `limit` | integer | 20 | 每页条数（最大 100） |
| `cursor` | string | — | 分页游标 |
| `sort` | string | `volume:desc` | 排序方式：`volume:desc`、`created_at:desc`、`close_time:asc` |

**响应 `200`：**

```json
{
  "data": [
    {
      "market_id": "0xabc123...",
      "title": "Will the LA Lakers win the 2026 NBA Championship?",
      "category": "sports",
      "tags": ["nba", "basketball"],
      "status": "active",
      "volume_24h": 45320.50,
      "volume_total": 1234567.89,
      "outcomes": [
        {
          "token_id": "217426331",
          "label": "Yes",
          "price": 0.12,
          "volume": 987654.32
        },
        {
          "token_id": "217426332",
          "label": "No",
          "price": 0.88,
          "volume": 246913.57
        }
      ],
      "resolution_source": "Polymarket",
      "close_time": "2026-06-30T23:59:59Z",
      "created_at": "2026-01-15T10:00:00Z"
    }
  ],
  "pagination": {
    "cursor": "eyJpZCI6IjB4YWJjMTIzIn0=",
    "has_more": true,
    "limit": 20
  }
}
```

## GET /markets/\{market_id\}

获取单个市场的完整详情。

**响应 `200`：**

```json
{
  "market_id": "0xabc123...",
  "title": "Will the LA Lakers win the 2026 NBA Championship?",
  "description": "This market resolves YES if the LA Lakers win the 2026 NBA Championship.",
  "category": "sports",
  "tags": ["nba", "basketball"],
  "status": "active",
  "volume_24h": 45320.50,
  "volume_total": 1234567.89,
  "liquidity": 98765.43,
  "outcomes": [...],
  "resolution_source": "Polymarket",
  "resolution_criteria": "...",
  "close_time": "2026-06-30T23:59:59Z",
  "created_at": "2026-01-15T10:00:00Z"
}
```

**错误码：**

| 错误码 | HTTP 状态 | 描述 |
|---|---|---|
| `MARKET_NOT_FOUND` | 404 | 市场 ID 不存在 |

## GET /markets/\{market_id\}/orderbook

获取某市场结果的实时 CLOB 订单簿。

**查询参数：**

| 参数 | 类型 | 默认值 | 描述 |
|---|---|---|---|
| `token_id` | string | — | 结果代币 ID（必填） |
| `depth` | integer | 10 | 每侧的价格档位数量 |

**响应 `200`：**

```json
{
  "market_id": "0xabc123...",
  "token_id": "217426331",
  "timestamp": "2026-03-19T14:22:05Z",
  "bids": [
    { "price": 0.11, "size": 500.0 },
    { "price": 0.10, "size": 1200.0 }
  ],
  "asks": [
    { "price": 0.12, "size": 300.0 },
    { "price": 0.13, "size": 800.0 }
  ],
  "mid": 0.115,
  "spread": 0.01
}
```

## GET /markets/\{market_id\}/trades

获取某市场结果的历史成交记录，默认按最新时间排序，支持分页。

**查询参数：**

| 参数 | 类型 | 默认值 | 描述 |
|---|---|---|---|
| `token_id` | string | — | 结果代币 ID |
| `limit` | integer | 50 | 每页条数（最大 100） |
| `cursor` | string | — | 分页游标 |
| `from` | ISO 8601 | — | 开始时间筛选 |
| `to` | ISO 8601 | — | 结束时间筛选 |

**响应 `200`：**

```json
{
  "data": [
    {
      "trade_id": "trade_xyz",
      "token_id": "217426331",
      "side": "BUY",
      "price": 0.12,
      "size": 100.0,
      "maker": "0xMakerAddress...",
      "taker": "0xTakerAddress...",
      "transaction_hash": "0xtxhash...",
      "timestamp": "2026-03-19T14:20:00Z"
    }
  ],
  "pagination": { "cursor": "...", "has_more": true, "limit": 50 }
}
```

---

## GET /markets/\{market_id\}/midpoint

获取某市场结果的当前中间价。

**查询参数：** `token_id`（必填）

**响应 `200`：**

```json
{
  "market_id": "0xabc123...",
  "token_id": "217426331",
  "mid": 0.115,
  "best_bid": 0.11,
  "best_ask": 0.12,
  "timestamp": "2026-03-19T14:22:05Z"
}
```

---

## GET /markets/search

对所有市场的标题和描述进行全文搜索。

**查询参数：**

| 参数 | 类型 | 描述 |
|---|---|---|
| `q` | string | 搜索关键词（必填，最少 2 个字符） |
| `limit` | integer | 每页结果数（默认 20，最大 50） |
| `cursor` | string | 分页游标 |

**响应 `200`：** 与 `GET /markets` 结构相同。

---

## WS /ws/markets/\{market_id\}

市场实时 WebSocket 流，在事件发生时推送订单簿更新和成交事件。

**连接地址：** `wss://api.broker.polymarket.com/ws/markets/\{market_id\}?token_id=\{token_id\}`

WebSocket 连接不计入每日 API 调用配额。

**消息格式：**

```json
{
  "type": "orderbook_update",
  "token_id": "217426331",
  "timestamp": "2026-03-19T14:22:05Z",
  "bids": [...],
  "asks": [...]
}
```

```json
{
  "type": "trade",
  "token_id": "217426331",
  "price": 0.12,
  "size": 100.0,
  "side": "BUY",
  "timestamp": "2026-03-19T14:22:06Z"
}
```
