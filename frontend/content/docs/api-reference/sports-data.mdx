# Sports Data

Base path: `/api/v1/data`

Query live and historical sports events traded on Polymarket, along with aggregated odds from 40+ traditional bookmakers. Use bias-opportunity endpoints to find markets where bookmaker consensus diverges from Polymarket pricing.

## Endpoints

| Method | Path | Description | Auth |
|---|---|---|---|
| `GET` | `/data/sports/categories` | Sport categories with active event counts | API Key |
| `GET` | `/data/sports/\{sport\}/events` | Events for a sport | API Key |
| `GET` | `/data/sports/\{sport\}/events/\{market_id\}/orderbook` | Orderbook for a sports event | API Key |
| `GET` | `/data/sports/\{sport\}/events/\{market_id\}/realized` | Resolved event result | API Key |
| `GET` | `/data/sports-odds/sports` | Sports with odds data available | API Key |
| `GET` | `/data/sports-odds/odds` | Aggregated odds across events | API Key |
| `GET` | `/data/sports-odds/odds/{event_id}` | Single event odds with bookmaker breakdown | API Key |
| `GET` | `/data/sports-odds/odds/\{event_id\}/bookmakers` | Full bookmaker detail for an event | API Key |
| `GET` | `/data/sports-odds/scores` | Live and recent scores | API Key |
| `GET` | `/data/sports-odds/bias-opportunities` | Events where bookmaker consensus differs from Polymarket | API Key |

## GET /data/sports/categories

Returns all sport categories with the number of currently active events in each.

**Example request:**

```bash
curl https://api.polydesk.io/api/v1/data/sports/categories \
  -H "X-API-Key: YOUR_KEY"
```

**Example response:**

```json
{
  "categories": [
    { "sport": "nba", "label": "NBA Basketball", "active_events": 42 },
    { "sport": "nfl", "label": "NFL Football", "active_events": 18 },
    { "sport": "mlb", "label": "MLB Baseball", "active_events": 31 }
  ]
}
```

## GET /data/sports/\{sport\}/events

Returns events for a given sport, with optional filtering and cursor-based pagination.

**Path parameters:**

| Param | Type | Description |
|---|---|---|
| `sport` | string | Sport slug (e.g. `nba`, `nfl`, `mlb`) |

**Query parameters:**

| Param | Type | Required | Description |
|---|---|---|---|
| `status` | string | No | Filter by status: `active`, `closed`, `resolved` |
| `limit` | integer | No | Max results per page (default 20, max 100) |
| `cursor` | string | No | Pagination cursor from previous response |

**Example request:**

```bash
curl "https://api.polydesk.io/api/v1/data/sports/nba/events?status=active&limit=10" \
  -H "X-API-Key: YOUR_KEY"
```

**Example response:**

```json
{
  "events": [
    {
      "market_id": "0x1234abcd",
      "title": "Will the Lakers win Game 5?",
      "sport": "nba",
      "status": "active",
      "end_date": "2026-04-15T03:00:00Z",
      "volume": 245000.50
    }
  ],
  "next_cursor": "eyJpZCI6MTAwfQ=="
}
```

## GET /data/sports/\{sport\}/events/\{market_id\}/orderbook

Returns the current orderbook for a sports event market.

**Path parameters:**

| Param | Type | Description |
|---|---|---|
| `sport` | string | Sport slug |
| `market_id` | string | Market identifier |

**Query parameters:**

| Param | Type | Required | Description |
|---|---|---|---|
| `token_id` | string | No | Specific token ID (YES or NO side). If omitted, returns both. |

**Example request:**

```bash
curl "https://api.polydesk.io/api/v1/data/sports/nba/events/0x1234abcd/orderbook?token_id=YES" \
  -H "X-API-Key: YOUR_KEY"
```

**Example response:**

```json
{
  "market_id": "0x1234abcd",
  "token_id": "YES",
  "midpoint": 0.62,
  "spread": 0.02,
  "bids": [
    { "price": 0.61, "size": 5200.00 },
    { "price": 0.60, "size": 12400.00 }
  ],
  "asks": [
    { "price": 0.63, "size": 3100.00 },
    { "price": 0.64, "size": 8700.00 }
  ],
  "timestamp": "2026-03-31T12:00:05Z"
}
```

## GET /data/sports-odds/scores

**Query parameters:**

| Param | Type | Required | Description |
|---|---|---|---|
| `sport` | string | No | Filter by sport key |

**Example request:**

```bash
curl "https://api.polydesk.io/api/v1/data/sports-odds/scores?sport=basketball_nba&completed=false" \
  -H "X-API-Key: YOUR_KEY"
```

**Example response:**

```json
{
  "scores": [
    {
      "event_id": "evt_abc123",
      "home_team": "Los Angeles Lakers",
      "away_team": "Boston Celtics",
      "home_score": 88,
      "away_score": 82,
      "period": "Q4",
      "clock": "3:42",
      "completed": false
    }
  ]
}
```

## GET /data/sports-odds/bias-opportunities

Finds events where the bookmaker consensus probability meaningfully differs from the current Polymarket price, indicating potential mispricing.

**Query parameters:**

| Param | Type | Required | Description |
|---|---|---|---|
| `min_bias_bps` | integer | No | Minimum bias in basis points (default 200 = 2%). Only return events where the absolute difference exceeds this threshold. |

**Example request:**

```bash
curl "https://api.polydesk.io/api/v1/data/sports-odds/bias-opportunities?min_bias_bps=500" \
  -H "X-API-Key: YOUR_KEY"
```

**Example response:**

```json
{
  "opportunities": [
    {
      "event_id": "evt_abc123",
      "title": "Lakers vs Celtics",
      "polymarket_price": 0.48,
      "bookmaker_consensus": 0.55,
      "bias_bps": 700,
      "direction": "underpriced_on_polymarket",
      "bookmaker_count": 41
    }
  ]
}
```
