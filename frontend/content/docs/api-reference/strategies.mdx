# Strategies API

Base path: `/api/v1/strategies`

Scope required: `strategies:execute`

The Strategies API exposes pre-built, executable trading strategies. v1 ships with convergence arbitrage only. Strategy logic is sourced from the Polymarket-Trading-Agents repository.

**Tier requirement:** Pro or Enterprise tier. Not available on Free tier.

## Available Strategies

| Slug | Description |
|---|---|
| `convergence` | Near-certain outcome markets (≥95% probability, ≤3 days to expiry) |

## Endpoints

| Method | Path | Description |
|---|---|---|
| `GET` | `/strategies` | List available strategies |
| `GET` | `/strategies/convergence/opportunities` | Markets meeting convergence criteria |
| `POST` | `/strategies/convergence/execute` | Place a convergence trade |
| `GET` | `/strategies/convergence/positions` | Active convergence positions |

## GET /strategies

List all available strategy slugs with metadata.

**Response `200`:**

```json
{
  "strategies": [
    {
      "slug": "convergence",
      "name": "Convergence Arbitrage",
      "description": "Trade near-certain outcome markets where implied probability is ≥95% and expiry is ≤3 days. These markets are approaching resolution at known prices but may still offer yield.",
      "tier_required": "pro",
      "risk_level": "low",
      "typical_return_bps": "50-200"
    }
  ]
}
```

## GET /strategies/convergence/opportunities

Find markets currently meeting convergence criteria: implied probability ≥ 95% with ≤ 3 days until market close.

**Query parameters:**

| Parameter | Type | Default | Description |
|---|---|---|---|
| `min_prob` | float | `0.95` | Minimum implied probability (0.01 to 0.99) |
| `max_days` | integer | `3` | Maximum days to expiry |
| `min_volume` | float | `1000` | Minimum 24h volume (USDC) |
| `category` | string | — | Filter by market category |

**Response `200`:**

```json

      "implied_probability": 0.96,

      "market_id": "0xdef456...",
```

## POST /strategies/convergence/execute

Place a convergence trade on a specific opportunity.

**Headers:** `X-API-Key: pm_live_sk_xxxxxxxxxxxxxxxx`

**Request body:**

```json
{
  "market_id": "0xabc123...",
  "token_id": "217426331",
  "size": 5000.0
}
```

The strategy validates that the opportunity still meets convergence criteria (≥95% prob, ≤3 days to expiry) at execution time. If criteria are no longer met, the request is rejected.

**Response `201`:**

```json
{
  "strategy_execution_id": "strat_exec_abc123",
  "strategy": "convergence",
  "order": {
    "order_id": "ord_abc123",
    "market_id": "0xabc123...",
    "token_id": "217426331",
    "side": "BUY",
    "type": "LIMIT",
    "price": 0.96,
    "size": 5000.0,
    "status": "OPEN",
    "broker_fee_bps": 5,
    "created_at": "2026-03-19T14:22:05Z"
  },
  "expected_profit": {
    "at_resolution": 208.33,
    "yield_bps": 417,
    "broker_fee": 12.50,
    "net_profit": 195.83
  }
}
```
