# Webhooks

Base path: `/api/v1/webhooks`

Receive real-time push notifications when events occur on your account or in markets you are tracking. Webhooks are available on the Pro tier. All deliveries are signed with HMAC-SHA256 using your webhook secret, sent in the `X-Polydesk-Signature` header.

## Endpoints

| Method | Path | Description | Auth |
|---|---|---|---|
| `POST` | `/webhooks` | Register a new webhook | API Key (Pro) |
| `GET` | `/webhooks` | List all webhooks | API Key (Pro) |
| `DELETE` | `/webhooks/{webhook_id}` | Delete a webhook | API Key (Pro) |

---

## Supported Event Types

| Event | Description |
|---|---|
| `order.placed` | An order was successfully placed |
| `order.filled` | An order was fully filled |
| `order.partially_filled` | An order was partially filled |
| `order.cancelled` | An order was cancelled |
| `position.closed` | A position was closed (market resolved or sold) |
| `price.alert` | A price alert threshold was crossed |
| `market.resolved` | A market you hold a position in has resolved |

---

## Signature Verification

Every webhook delivery includes an `X-Polydesk-Signature` header containing an HMAC-SHA256 hex digest of the raw request body, signed with your webhook secret.

```python
import hmac
import hashlib

def verify_signature(body: bytes, signature: str, secret: str) -> bool:
    expected = hmac.new(secret.encode(), body, hashlib.sha256).hexdigest()
    return hmac.compare_digest(expected, signature)
```

## POST /webhooks

Register a new webhook endpoint to receive event notifications.

**Request body:**

```json
{
  "url": "https://example.com/hooks/polydesk",
  "events": ["order.filled", "position.closed", "market.resolved"],
  "secret": "whsec_your_secret_key_here"
}
```

| Field | Type | Required | Description |
|---|---|---|---|
| `url` | string | Yes | HTTPS URL to receive webhook POST deliveries |
| `events` | string[] | Yes | Array of event types to subscribe to |
| `secret` | string | Yes | Secret key used for HMAC-SHA256 signature verification |

**Example request:**

```bash
curl -X POST https://api.polydesk.io/api/v1/webhooks \
  -H "X-API-Key: YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://example.com/hooks/polydesk",
    "events": ["order.filled", "position.closed"],
    "secret": "whsec_abc123secret"
  }'
```

**Example response:**

```json
{
  "webhook_id": "wh_abc123",
  "url": "https://example.com/hooks/polydesk",
  "events": ["order.filled", "position.closed"],
  "status": "active",
  "created_at": "2026-03-31T12:00:00Z"
}
```

## Webhook Delivery Format

When an event occurs, Polydesk sends a `POST` request to your URL with the following body structure:

```json
{
  "event": "order.filled",
  "webhook_id": "wh_abc123",
  "timestamp": "2026-03-31T12:05:00Z",
  "data": {
    "order_id": "ord_xyz789",
    "market_id": "0x1234abcd",
    "side": "YES",
    "price": 0.62,
    "size": 500.00,
    "filled_at": "2026-03-31T12:05:00Z"
  }
}
```

Polydesk retries failed deliveries (non-2xx responses) up to 5 times with exponential backoff (10s, 30s, 90s, 270s, 810s). After 5 failures, the webhook is marked as `degraded`.

## GET /webhooks

Returns all registered webhooks for your account.

**Example request:**

```bash
curl https://api.polydesk.io/api/v1/webhooks \
  -H "X-API-Key: YOUR_KEY"
```

**Example response:**

```json
{
  "webhooks": [
    {
      "webhook_id": "wh_abc123",
      "url": "https://example.com/hooks/polydesk",
      "events": ["order.filled", "position.closed"],
      "status": "active",
      "created_at": "2026-03-31T12:00:00Z"
    },
    {
      "webhook_id": "wh_def456",
      "url": "https://example.com/hooks/alerts",
      "events": ["price.alert"],
      "status": "degraded",
      "created_at": "2026-03-20T08:00:00Z"
    }
  ]
}
```

## DELETE /webhooks/\{webhook_id\}

Deletes a webhook. No further deliveries will be sent to this endpoint.

**Path parameters:**

| Param | Type | Description |
|---|---|---|
| `webhook_id` | string | Webhook identifier |

**Example request:**

```bash
curl -X DELETE https://api.polydesk.io/api/v1/webhooks/wh_def456 \
  -H "X-API-Key: YOUR_KEY"
```

**Example response:**

```json
{
  "webhook_id": "wh_def456",
  "deleted": true
}
```
