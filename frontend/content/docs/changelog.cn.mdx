# 更新日志

## v1.0.0 — 2026 年 3 月

Polydesk API 正式公开发布。

### 发布功能

**身份认证**
- 邮箱/密码注册与登录
- 基于钱包的非托管认证（EIP-191 挑战/验证流程）
- 具有细粒度权限范围的 API 密钥管理
- JWT 访问令牌（15 分钟有效期）+ 不透明刷新令牌（30 天有效期）

**市场**
- 支持按类别、状态、标签过滤的完整市场列表
- 市场详情、订单簿（orderbook）、历史成交、中间价
- 全文搜索
- 实时 WebSocket 订单簿流

**订单**
- 托管模式：由 Broker 代为签名并提交
- 非托管模式：EIP-712 构建/提交流程
- 订单类型：LIMIT（限价）、MARKET（市价，FOK）、GTD（含到期时间）、TWAP
- 列出、查询、取消单个订单，取消所有开放订单

**投资组合**
- 含未实现盈亏的持仓信息
- USDC 余额（总计、已锁定、可用）
- 盈亏明细，包含已实现/未实现拆分及手续费明细

**数据：体育**
- 145 个体育及电竞类别
- 赛事列表、历史订单簿（完整时间序列）、已实现成交

**数据：NBA**
- 与 Polymarket 市场关联的直播比赛列表
- 含实时比分、节次和剩余时间的比赛详情
- 融合数据（fusion）接口：实时比分 × Polymarket 赔率 + 偏差信号（bias signal）
- 实时 WebSocket 同步

**数据：BTC**
- 多时间框架（timeframe）预测市场数据（5m、15m、1h、4h）
- Polygon 链上（on-chain）成交数据，精度达毫秒级
- 通过 Gamma API 进行历史数据回填

**分析**
- 基于 AI 的市场概率估算（DeepSeek-R1）
- 全市场扫描以寻找定价偏差机会
- NBA 比赛 AI 方向性建议
- 自然语言查询接口

**策略**
- 收敛套利（convergence arbitrage）：发现并执行接近确定结果的交易
- 活跃收敛仓位追踪

**频率限制**
- 免费版：每日 500 次调用，每分钟 20 次突发请求，每日 10 次 AI 调用
- 专业版（每月 99 美元）：无限次调用，每分钟 300 次突发请求，无限次 AI 调用
- 企业版：定制方案

**Webhooks**
- 事件类型：`order.filled`、`order.cancelled`、`market.resolved`、`position.opened`、`position.closed`、`strategy.executed`、`analysis.signal`
- HMAC-SHA256 签名载荷
- 指数退避重试（最多 5 次）
